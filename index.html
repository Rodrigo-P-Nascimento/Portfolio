    Deterministic Finite Automaton Minimize

    This was a project for the Theory of Computation course, and the group consisted of me and my colleague EgÃ­dio Neto. The code written in C implements a minimization algorithm for a Deterministic Finite Automaton (DFA). The code takes as input the description of the DFA and then uses minimization techniques, such as the Myhill-Nerode algorithm or the Table Filling Method, to reduce the number of states and transitions in the DFA. During the execution of the code, detailed steps of the minimization process will be displayed. The final result will be a minimized DFA, which can be used to optimize the representation and performance of the automaton.

    Optimization of product production lines

    This project aims to find the best production configuration (minimizing time) for a set of products and production lines, using input data from a file. Initially, we employ a greedy heuristic to generate an initial solution. Next, we apply the Variable Neighborhood Descent (VND) algorithm with two neighborhood moves to refine this solution and achieve an even better result. The project was developed in collaboration with my fellow undergraduate students, Bertrand Dantas and JoÃ£o Victor, as part of the Algorithm Analysis and Design course. We are proud to share the outcome of our public work, which represents our combined effort to solve this challenging optimization problem.
